Smart Parking System with ESP8266 & RFID

IoT-based parking management — real-time slot monitoring, RFID access, MySQL backend and automated gates.

🚀 Project Overview

Made with ❤️ by Emmanuel TUYISENGE
DEMO VIDEO GitHub Code

This project monitors parking slots using infrared sensors, controls gates via servos, and authenticates vehicles with RFID cards. The ESP8266 sends and receives data to a PHP/MySQL backend to store transactions and sensor states.

Key Features

🔐 RFID-based access (entry & exit)
📶 ESP8266 WiFi connectivity
🚦 Live slot monitoring (IR sensors)
💾 MySQL & PHP backend
🔁 Automated gate control
💳 Payment & balance checking

🔥 Safety & Environment Monitoring

The system is not limited to parking management. It also integrates fire detection and air quality monitoring for improved safety inside the parking lot. Using sensors such as MQ-2/MQ-135 for smoke and harmful gases and temperature/IR sensors for fire detection, the system can:

  • 🚨 Detect fire, smoke, or dangerous gas levels in real-time.
  • 🌬️ Trigger exhaust fans for bad air or smoke.
  • 💦 Activate sprinklers or fire extinguishers automatically in case of fire.
  • 📡 Send alerts to admins and store events in the database for monitoring.

This ensures the parking area is safe, eco-friendly, and responsive to emergencies without human intervention.
